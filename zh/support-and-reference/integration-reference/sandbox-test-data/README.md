当你需要以下内容时，从这里开始：

* 运行稳定的沙箱测试场景

* 使用沙箱验证或 UAT 准备的共享输入

#### 沙箱测试数据应该用于什么？

使用本部分获取稳定的沙箱航线、测试卡和其他验证期间使用的已发布输入。

#### 何时应查看沙箱测试数据？

在沙箱验证、脚本测试或 UAT 准备之前，当你需要已知良好的测试输入时查看。
