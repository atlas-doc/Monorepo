# 沙箱测试航线

{% include "../../../.gitbook/includes/eva-help-hint.md" %}

当你需要稳定的沙箱搜索和预订场景时，使用这些航线。

### 如何使用本页面

* 选择你想要测试的航空公司所支持的航线
* 使用沙箱凭证和沙箱端点
* 将航线可用性视为测试指南，而非生产覆盖范围
