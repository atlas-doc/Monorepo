# 产品指南

使用此部分可在实现端点详情之前选择正确的指南。

浏览产品指南的三个部分：

### 预订

此部分涵盖完整的预订领域。

从[预订](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/Ac99iL6Dsljikm4FF7ub)开始。

然后使用：

* [预订概述](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/82DaHlpWfsy0ANSplNI3)了解共享规则和标识符
* [预订流程](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/wkGCRQidKhCerhUNdoz7)选择端到端预订路径
* [预订步骤](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/7Hivwro5sZdOotzmaEIn)了解各步骤预订页面
* [可选附加服务](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/Bbuhdlzobxaya4FUurUl)了解可选的座位和行李

### 预订后

此部分涵盖退款、作废、订单维护和 PNR 跟进。

从[预订后](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/R6rkbsTfrs5gZzElecNB)开始。

### 扩展与集成

此部分涵盖通知、数据源和特殊集成。

从[扩展与集成](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/SdBLDY5Oi5LsKnOOYM4C)、[Webhook 概述](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/1kDgZtL5TYuOLUmAxRd6)或[航班数据源](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/lwbbbCSVAlYcqOhIybiX)开始。

### 需要精确字段时请使用 API 参考

需要精确的架构、参数和端点级示例时，请使用[API 参考](/broken/spaces/6LsKtmbJhZxgxraY5mHB/pages/LfT8Y3jMIGXTnxwihZhV)。

### 需要集成步骤时请使用集成指南

需要沙箱设置、开发、UAT 和生产上线时，请使用[集成指南](https://app.gitbook.com/s/6LsKtmbJhZxgxraY5mHB/)。
