# 预订步骤指南

使用此部分获取跨多个预订流程使用的步骤级参考页面。

这些页面在每个预订流程中并非固定的顺序。

当您需要端到端路径时，请先使用流程页面。

在您已经知道要构建哪个预订流程后，使用此部分。

当您需要以下内容时从这里开始：

* 详细了解一个预订步骤
* 将任务映射到正确的端点序列
* 构建或排查搜索、验证、订单、支付或订单跟进
